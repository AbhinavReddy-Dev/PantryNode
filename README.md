## Installation

- All the `code` required to get started

### Clone

- Clone this repo to your local machine using `git clone https://github.com/ChicoState/PantryNode`

```shell
$ cd PantryNode
```

### Setup

> now install npm packages

```shell
$ npm install
```

> now start the server

```shell
$ npm start
```

> Alternative to `npm start`

```shell
$ npm install nodemon
```

> Run Nodemom

```shell
$ nodemon bin/www
```

> Brower

```shell
  http://localhost:3000/
```
